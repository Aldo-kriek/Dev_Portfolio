Vehicle Management System

  • Category: Web Application
  • Tools: HTML, CSS, JavaScript, PHP, SQL
  • Project date: 16 November, 2022
  • Project URL: www.example.com


Description

Calculator consists of numbers and mathematical operators that can be used to perform basic calculations and then display these answers on a screen. Number values can be stored to a memory file and then later re-called for use in calculations. The memory file can be cleared and contents can be removed from the storage file. A clear button can be used to clear contents that are currently active on the screen.



Key Features

  • Numbers inputs entered and then viewed on a screen.
  • Performs multiplication, division, addition and subtraction.
  • Equals sign performs calculations and displays answers on a screen.
  • On-screen values can be stored to memory and then re-called for later usage.
  • Values can be removed from the Memory storage file.
