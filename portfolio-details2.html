Property Management Maintenance

  • Category: Software Application
  • Tools: Java & SQL
  • Project date: 8 September, 2022
  • Project URL: Property Management


Description

Assign tasks and services to an employee so that they can assist tenants with various property repair maintenance tasks and issues. Monitor and manage progress of the status of current projects and display information about tenant and employee status.



Key Features

  • Home Screen with fast entry access to application.
  • Registration screen to assign tasks to volunteers.
  • Registration screen to allow tenants to report faults.
  • Results screen containing table view to view repair progress and job status.
  • Combo box to filter and search for user data .
  • Create, update, View and delete entries from database table .


Conclusion

Application was created to allow property agencies to manage tenant requests and assign them to volunteers to monitor and maintan the outcome of the repair.
